# External Feature Matrix

This is a clean-room feature map for Vantix. It records external product patterns that are useful to reproduce with original implementation work.

## White-Box Workflow Patterns
- White-box plus live-target workflow
- Resumable workspaces
- Proof-only findings
- Source-aware attack-path generation
- Prompt and workspace artifact capture

## Agentic Platform Patterns
- Persistent `Flow` / `Task` / `Action` model
- API-first backend
- Vector-ready memory model
- Specialist-role orchestration
- Frontend-ready service boundaries

## Existing strengths preserved here
- Local `cve-search` and `searchsploit` pipeline
- Structured self-learning in `memory/learning/*.jsonl`
- Mode-specific guardrails for `CTF` and `KoTH`
- Existing shell/Python operator tooling
